# Plugin and Connector SDK

## Required interfaces

- Discover assets.
- Collect configuration.
- Collect or reference metrics and logs.
- Subscribe to events.
- Query evidence on demand.
- Health and capability report.
- Checkpoint and resume.
- Normalize source identifiers.
- Declare permissions and rate limits.

## SDK deliverables

Python package, interface specification, example connector, simulator, fixtures, contract test suite, packaging standard, compatibility matrix, and developer documentation.
