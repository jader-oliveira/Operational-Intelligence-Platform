# Roadmap

| Phase | Name | Scope | Indicative duration |
| --- | --- | --- | --- |
| 0 | Foundation | Architecture, schemas, security, labs, baseline metrics, ADRs | 6-8 weeks |
| 1 | Observe | Connector SDK, VMware connector, inventory, events, configuration, coverage | 8-12 weeks |
| 2 | Understand | Data platform, dependency graph, ownership, service mapping | 8-12 weeks |
| 3 | Investigate | Evidence engine, incident workspace, RAG, reasoner, challenge review | 10-14 weeks |
| 4 | Predict and Guard | Forecasting, anomaly, drift, hardening, security posture | 10-14 weeks |
| 5 | Advise and Approve | Remediation, policy, approvals, orchestration, verification | 10-14 weeks |
| 6 | Learn and Scale | Fine-tuning, operations DNA, governance dashboards, rollout | Ongoing |

Durations are planning ranges, not promises. Parallel workstreams are possible after Phase 0, but the canonical data model, security boundaries, evaluation baseline, and approval model must not be skipped.
