# Disaster Recovery

## Protect

PostgreSQL, graph data or projections, object storage, event offsets, model registry, policy repository, audit store, Git repositories, and encryption material.

## Requirements

- Document RPO and RTO by component.
- Automated backups with immutable copies.
- Quarterly restore tests.
- Rebuildable stateless services.
- Versioned model and dataset artifacts.
- Offline copy of critical runbooks and architecture.
- Tested degraded mode in which the platform cannot automate changes.
