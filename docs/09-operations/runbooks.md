# Platform Runbooks

Required runbooks:

- Connector credential failure.
- Source API throttling.
- Event backlog.
- Schema incompatibility.
- Metrics or log-store outage.
- Graph projection delay.
- Retrieval quality degradation.
- Model server outage.
- Unsafe or unsupported AI output.
- Approval workflow stuck.
- Automation execution failure.
- Rollback failure.
- Audit pipeline failure.
- Restore from backup.
- Disable a model, connector, or execution adapter.
