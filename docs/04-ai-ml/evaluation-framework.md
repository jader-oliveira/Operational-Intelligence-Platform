# Evaluation Framework

## Golden evaluation suites

- Known incidents with verified root cause.
- Configuration and security findings with expert labels.
- Change plans with known safe and unsafe variants.
- Forecasting backtests.
- Retrieval relevance judgments.
- Tool-use and permission tests.
- Adversarial and ambiguous cases.

## Core metrics

- Root-cause top-1 and top-3 accuracy.
- Evidence precision and recall.
- Citation correctness.
- Unsupported-claim rate.
- Recommendation safety and completeness.
- Human acceptance and correctness.
- Structured-output success.
- Calibration.
- Latency and cost.
- Regression against previous production model.

## Online evaluation

Shadow recommendations, engineer feedback, outcome validation, rollback rate, incident recurrence, and sampled expert review.
