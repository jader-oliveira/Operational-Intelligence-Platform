# Platform Observability

## Platform telemetry

- OpenTelemetry traces across API, workflow, retrieval, tools, and model calls.
- Prometheus metrics for services, queues, data freshness, models, and workflows.
- Structured logs with correlation identifiers.
- Grafana dashboards and alerting.
- Audit records separated from operational logs.

## AI-specific dashboards

- Issues detected.
- Root-cause accuracy.
- Recommendations generated, approved, rejected, executed, successful, failed, and rolled back.
- Configuration and security findings.
- Evidence coverage.
- Model comparison.
- Latency, tokens, GPU utilization, and cost.
- Trust-score distribution.
- Drift and regression alerts.
